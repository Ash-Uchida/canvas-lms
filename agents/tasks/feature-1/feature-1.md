# Feature 1: Dashboard Card Appearance Preset Selector

## One-Line Problem Framing
Canvas users need a simple way to personalize Dashboard Card View appearance with approved presets that persist per user across sessions without breaking existing dashboard behavior.

## Scope Snapshot
- Add a preset selector in Dashboard Card View.
- Persist the selected preset at user scope.
- Apply the saved preset automatically on return.
- Fall back safely to `classic` when preference is missing or invalid.

## Out of Scope
- Freeform custom CSS or uploaded visual themes.
- Full dashboard layout redesign.
- Mobile app parity for this lab.
